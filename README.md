## 🛠️ Tools & Libraries Used
- Python 3
- Google Colab
- scikit-learn
- pandas, numpy
- matplotlib

## 🌸 Dataset
All experiments use the **Iris Dataset** from the UCI ML Repository.  
- 150 samples, 4 features, 3 classes (Setosa, Versicolor, Virginica)
- Loaded directly via `sklearn.datasets.load_iris()`
